Exercises

Server, OSC, Synth, Node, Bus, Buffer

  • Find a way to list all servers declared by the system in the Server class.
  • Create a Synth
  • List the current node tree of the default server on the post window
  • Explain the structure of the node tree of the default server.
  • Load a buffer from a sound file in supercollider. Find its number and its frame size and number of audio channels. Play the file in supercollider
  • Busses:
    • Control busses
      • Create a control-rate bus.
      • Write to it using a control-rate ugen.
      • Map a control
    • Audio busses
    • Multichannel expansion

Unit Generators, SynthDefs, MouseX/Y, Envelopes.

  • Map the output range of a Unit Generator's signal to the range from 2 to 25 in two different ways:
    • By modifying the corresponding parameters already built-in the Unit Generator.
    • By using a single linear mapping method applied to the Unit Generator.
  • Create a SynthDef with two control inputs, and code this to create the control inputs in two different ways:
    • Implicitly
    • Explicitly
  • Answers
    • Explicit
    • Implicit
      • Control one parameter of a synth with mouse x (horizontally) and another parameter with mouse y (vertically)
      • Define a SynthDef that uses a percussive envelope and create synths of different durations from it

Synthesis Techniques

Non-sample based

  • Demonstrate additive synthesis of tones with a small number of sinusoidal components
  • Demonstrate additive synthesis using wavetables and sinusoids (Blip)
  • Answer 1: Simple Wavetable
  • Answer 2: Blip
    • Demonstrate use of Delay
    • Use a Comb Filter to generate pitched sound signals
    • Demonstrate use of Hi-pass, low-pass, band-pass and band-reject filters
    • Use Formants to generate sounds resembling human vocalisation
